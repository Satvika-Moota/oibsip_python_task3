A modern and user-friendly **Secure Password Generator** built with **Python and Tkinter**.
This application helps users generate strong, random, and customizable passwords to improve online security.

## Features
* Clean dark modern user interface
* Choose password length
* Options to include:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Symbols
* Option to exclude similar characters (O, 0, l, 1)
* One-click copy to clipboard
* Error handling for invalid inputs

##  Technologies Used
* Python
* Tkinter
* Random & String modules
* Pyperclip (for copying password)

##  Installation
1. Clone the repository:
```bash
git clone https://github.com/Satvika-Moota/oibsip_python_task3
```

2. Navigate to the project folder:
```bash
cd Secure-Password-Generator
```

3. Install required library:
```bash
pip install pyperclip
```

4. Run the application:
```bash
python main.py
```
##  How to Use
1. Enter desired password length
2. Select the character types you want
3. Click **Generate Password**
4. Copy it using **Copy to Clipboard**
